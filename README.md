## PWA support

This app now includes Progressive Web App essentials (manifest + service worker), so users can install it to the home screen from supported browsers (including Safari "Add to Home Screen").

- Build normally: `npm run build`
- Serve over HTTPS in production
- On Safari/iOS, open the app URL and choose **Share → Add to Home Screen**
